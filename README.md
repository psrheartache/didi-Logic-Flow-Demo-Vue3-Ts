# didi-Logic-Flow-Demo-Vue3-Ts

一个基于滴滴团队Login-Flow流程图编辑库实现的Vue3+Vite+TypeScript的MvpDemo

## 项目运行

```
1. yarn or npm i
```

```
2. yarn dev  or  npm run dev
```
